# Tech Stack and Commands

## Tech Stack
*   **Frontend Framework:** React
*   **Language:** TypeScript
*   **Build Tool:** Vite
*   **Linting:** ESLint
*   **Package Manager:** npm

## Commands

### Development Commands
*   `npm run dev`: Starts the development server using Vite.
*   `npm run build`: Builds the project for production, first compiling TypeScript (`tsc -b`) and then using Vite for the final build.
*   `npm run lint`: Runs ESLint across the codebase to check for code style and potential errors.
*   `npm run preview`: Serves the production build locally for previewing.

### System Utility Commands (Windows)
*   `cd <directory>`: Change directory.
*   `dir`: Lists files and subdirectories in the current directory.
*   `type <filename>`: Displays the content of a text file.
*   `findstr <pattern> <files>`: Searches for text patterns in files (similar to `grep`).
*   `git <command>`: Version control commands.
